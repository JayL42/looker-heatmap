Looker Custom Heatmap Visualization

This page hosts a custom heatmap visualization for Looker.

Visualization URL:

Setup in Looker

  1. Go to Admin > Visualizations
  2. Click Add Visualization
  3. Enter:
    • ID: custom_heatmap
    • Label: Custom Heatmap
    • Main: Copy the URL above
  4. Click Save

Usage

In any Explore, structure your query as:

  • First dimension: X-axis categories (columns)
  • Second dimension: Y-axis categories (rows)
  • First measure: Value for color intensity

Then select Custom Heatmap from the visualization picker.

Features

  • 6 built-in color schemes + custom colors
  • Configurable cell padding, values, and formatting
  • Axis label controls with rotation
  • Gradient legend with positioning options
  • Tooltips and drill-through support
